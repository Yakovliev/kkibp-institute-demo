# Сайт Київського кооперативного інституту бізнесу і права

Статична демоверсія сайту інституту. Поточна кодова база підготовлена на основі готової демоверсії коледжу, тому перший етап роботи — контрольовано замінити коледжну структуру, тексти, метадані та документацію на інститутські.

## Що є в комплекті

- `index.html` — головна сторінка.
- `institute.html`, `admissions.html`, `students.html`, `alumni.html`, `science.html`, `library.html`, `news.html` — сторінки, успадковані з демоверсії коледжу і готові до адаптації.
- `css/styles.css` — стилі, адаптивні брейкпоінти та теми день/ніч.
- `js/main.js` — навігація, мобільне меню, пошук, перемикач теми, рендеринг новин і допоміжні інтерактивні елементи.
- `js/news-data.js` — спільне джерело даних новин.
- `assets/` — локальні зображення, логотипи, іконки та шрифти.
- `docs/college-docs/` — архів документації, рішень і скріншотів коледжної демоверсії для використання як основи майбутньої інститутської документації.

## Запуск

Сайт можна відкрити подвійним кліком по `index.html`. Для коректної локальної розробки зручніше запустити простий сервер:

```bash
python3 -m http.server 8000
```

Після цього відкрити `http://localhost:8000`.

## Перевірка

У репозиторії є Playwright smoke-тести для базової перевірки сторінок, навігації та адаптивності:

```bash
npm run test:playwright
```

## Поточний статус

Репозиторій уже прив'язаний до `Yakovliev/kkibp-institute-demo`. Коледжні матеріали збережені як референс у `docs/college-docs/`; сторінки сайту, соціальні метадані та частина допоміжних скриптів ще потребують окремої інститутської адаптації.
